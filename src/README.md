# CatalogPRO v2

Visualizador de laminas para LOMHIFAR S.L.

## Concepto

La unidad atomica es la **lamina** (no el articulo). Los catalogos son colecciones ordenadas de laminas.

## Stack

- Backend: Node.js + Express + TypeScript + PostgreSQL
- Frontend: HTML + CSS + JavaScript vanilla (tablet vertical, rosa LOMHIFAR)
- Despliegue: Railway

## Tipos de catalogo

1. **Maestro** - el catalogo completo que tu (admin) montas
2. **Clon** - copia personalizada para cada comercial
3. **Express** - mini-catalogos para campanas/novedades (vinculados al maestro)
